    Delete an object or a bucket

    Delete an object¤

    1. To delete an object, click the check box of the object you want to delete.

    2. Under ACTIONS select Delete.

    3. Review the objects you want to delete and select Delete.

    Delete a bucket¤

    1. To delete a bucket, click the ellipsis next to the bucket you want to delete.

    2. Select Delete.

    3. Confirm the deletion of the bucket you want to delete and select Delete.
